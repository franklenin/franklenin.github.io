An insurance claim analysis system that uses AI agents to automatically process and verify claims. The system analyzes claim documents through fraud detection, optical character recognition, and intelligent decision-making using local LLMs (Ollama).


## Features

- **Fraud Detection (Veritas)**: Verify image authenticity before processing
- **Document Analysis**: Extract text from images and documents using OCR with vision LLMs (`qwen3-vl:8b`)
- **AI-Powered Decisions (Atlas)**: Use local LLMs (`llama3.1`) to analyze and approve/deny claims
